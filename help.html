                                                   A propos du Projet

Notre projet est un web service qui consiste a superviser les serveurs on utilisant leur adresse ip et port

Ce web service permet de voir le statut serveur la ram utilisé le cpu et le temp de reponse

Il permet aussi de gerer les users pour ajouter un simple user il faut demander au super admin

Finalement le super admin peut consulter l'historique d'un user afin de controller l'acces
